母婴电商网站 - 个人毕业设计作品
============================

----------------------------

应用基于 [Yii 2](http://www.yiiframework.com/) Basic 开发。

目录结构
-------------------

      assets/             contains assets definition
      commands/           命令行命令
      config/             应用配置文件
      controllers/        控制器类
      mail/               e-mails视图文件
      models/             数据模型类
      runtime/            运行时的生成文件
      tests/              测试文件
      vendor/             第三方依赖包
      views/              视图文件
      web/                包含入口脚本和资源文件





安装
------------

### 通过Composer安装依赖

如果没有安装 [Composer](http://getcomposer.org/), 请参照 [getcomposer.org](http://getcomposer.org/doc/00-intro.md#installation-nix).

使用以下命令安装应用依赖：

~~~
composer install
~~~
